Занимался развертыванием, настройкой сервера на основе операционной системы unix, под геопортал открытых данных Никопольского городского совета. Создание raid массивов, настройка и администрирование сетевых хранилищ.

Cетевое хранилище - основным предназначением является предоставление служб для хранения данных на файловом уровне, другим устройствам в сети. Cетевые хранилища имеют консольный интерфейс. Позволяют использовать такое решение как кластеры для высоко нагруженных приложений, обеспечивают надёжность хранения данных.

криптография


Использую сертификаты и шифрованные ключи для сетевого оборудывания, это позволяет позволяет автоматизировать, ускорить а главное максимально защитить оборудование и данные сети. Криптография позволяет хранить информацию в сети существенно снизив уровень компрометации данных. Использую открытые библиотеки openSSL и ...

bash

Использую в работе с unix и linux командный процессор, который позволяет выполнять команды скриптов, что упрощает и автоматизирует рутинные действия и настройки серверной части. Он поддерживает авто дополнение имён файлов и каталогов, подстановку вывода результата команд, переменные, контроль над порядком выполнения, операторы ветвления и цикла. Также позволяет создавать скрипты осуществляющие контроль над управляемыми сетевыми коммутаторами, функции перезагрузки, сканирования сети ...
